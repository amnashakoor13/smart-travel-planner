🧱 TECHNOLOGY STACK (FINAL)

Backend: Node.js + Express.js

Database: MongoDB

APIs Integrated:

Maps API → Location & route visualization

Weather API → Real-time weather & forecasts

AI API → Buddy Bot & smart recommendations
